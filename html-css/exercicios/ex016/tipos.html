Alturas (itens métricos) altura x (altura base das minusculas é do tamanho da letra x) altura das maiusculas (altura que vai da base x até uma altura maxima) ascendente descendente corpo (somatorio de todas as alturas)

Itens anatomicos (geometricos) serifa (risquinho no final de algumas letras de algumas fontes para dar a ideia de uma linha imaginaria para facilitar a leitura das palavra) hastes (parte vertical reta) filetes (parte horizontal que liga duas hastes ou arcos) arcos (parte curvada na parte de cima ou de baixo) esporção( ponta na parte de baixo, uma especie de base como se estivesse apoiando a letra, parece um salto-alto) vertice (ponta na parte de cima) terminal (como a parte lateral do "r", um vertice que não liga duas partes) anatomicos (corporais) braço (sai de uma haste e vai para cima na diagonal) perna (sai de uma haste e vai para baixo na diagonal) pé (base da letra) espinha (só existe no "s" curvas ) barriga (arco dois arcos juntos)
